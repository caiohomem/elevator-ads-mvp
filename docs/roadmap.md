# Roadmap

1. Bootstrap repository: create the initial backend, frontend, and documentation structure.
2. Building management: manage buildings that host elevator screens.
3. Screen management: register and manage elevator screens and their status.
4. Advertiser management: manage advertiser accounts and basic profile data.
5. Campaign management: define campaigns, budgets, targeting, and schedules.
6. Creative management: upload, review, and approve ad creatives.
7. Next-ad decision endpoint: select the next ad to play for a screen request.
8. Proof-of-play tracking: record ad playback events from elevator screens.
9. Basic reports: expose campaign and playback reporting.
10. SSP inventory model: represent available elevator screen inventory for supply-side workflows.
11. DSP campaign model: represent demand-side campaign configuration.
12. Auction engine: choose ads through auction logic.
13. OpenRTB adapter: add OpenRTB compatibility after the core system exists.
