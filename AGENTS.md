# Agent Rules

- Work only on the requested issue.
- Do not implement unrelated features.
- Do not commit secrets.
- Do not add authentication unless requested.
- Do not add OpenRTB yet.
- Do not add SSP/DSP logic yet.
- Do not implement real-time ad serving (next-ad endpoints, auction, DSP/SSP bidding) unless explicitly requested.
- The first delivery model is scheduled DOOH playlist delivery; implement that before any real-time mechanism.
- Keep PRs small and focused.
- Run tests/build before finishing.
